# 미션 - 개발자 비상근무

### 구현할 목록
- 월 및 시작 요일을 입력받는 기능
- 평일 비상 근무 순번 입력받는 기능
- 휴일 비상 근무 순번 입력받는 기능
- 전체 근무 일지 출력 기능


### Model
- Month
	- 월 및 시작 날짜를 관리
	- 월을 입력 받으면 날짜에 맞는 요일을 생성
	- 1~12월이 아니면 에러
	- 지정된 요일이 아니면 에러

- 평일 비상 근무 순서 // 이름 미정
	- 이름이 중복되면 에러
	- 최소 5명은 넘어야 함
	- 이름이 0~5자를 넘기면 에러

-  휴일 근무 순서 // 이름 미정
	- 이름이 중복되면 에러
	- 최소 5명은 넘어야 함
	- 이름이 0~5자를 넘기면 에러
